## Getting Started 🚀

### Prerequisites
- Git
- Node.js / Python (depending on backend)
- Code editor like Cursor or VS Code

### Installation

1. Clone the repository
```
git clone https://github.com/VarunKarthikB-18/Smart-Voting-App.git
cd Smart-Voting-App
```

2. Install dependencies  
(Example for Node.js)
```
npm install
```

3. Run the app  
(Example)
```
npm start
```
or if Python backend:
```
python app.py
```

4. Access the app at:
```
http://localhost:3000
```
